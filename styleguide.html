Style-guide.

Inspired by the style-guides of Chen Hui Jing, Dana Byerly, and Bryan Braun, this page illustrates how HTML elements are displayed on DuncanRitchie.co.uk with my CSS.

Elements look worse here than on other pages, which suggests that I should simplify my CSS so it’s easier to keep the styles of my website consistent! Elements that are not used on this site are simply taking on your browser’s default styles — but the defaults also apply to first-level headings, because outside of this page <h1> elements are visually hidden, shown only to screen-reading software.
